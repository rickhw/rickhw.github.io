變數

宣告

bash 的變數不需要宣告型別，直接宣告即可使用，但是要注意幾點：

  1. 等號 (Assignment) 兩邊不可以用空格，name = rick 這樣會報錯。
  2. 值 (Value) 的雙引號可有可無，name="rick" 和 name=rick 都可以。
    • 沒有引號的值不允許有空白。
  3. 變數名稱只能是 [A-Za-z0-9_] 這樣的格式，而且不能以數字開頭。
    • 注意：不可以使用 dash - 當作變數名稱，- 在 bash 有其他意義。
  4. 避免使用 env 裡的字，建議都使用特殊的 prefix 當作自己的 scope，避免衝突。
