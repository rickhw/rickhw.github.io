Serverless All-Star in 2018 (活動預告)

原定時間是 2017/12/07 ，後來改到 2018/03/29，活動名稱改為 Serverless All-Star，我的主題沒有改變: Ops as Code using Serverless，歡迎大家來彭場！

這場演講的 Slide 請參考 Serverless All-Star in 2018 (Public Slide)


內容會以 Ops as Code 為主軸，這也是這幾年工作的心得與想法，同時如何利用 Serverless 的概念作應用。

主軸如下：

  • Ops: 專業領域、經典問題、SRE
    • 系統管理 (Sys Admin)、系統維運 (System Operations)
    • 跟監控有關的問題，因為有另位講者會講，所以我只會帶過。
    • 想知道我怎麼做監控的，請參閱去年的分享： 淺談系統監控與 CloudWatch 的應用 - AWS User Group Taiwan
  • Code: 軟體工程、基本功
  • Serverless: 新時代的技術、與概念
  • Connect: 把這些連結起來。

對於這樣主題有興趣的朋友，歡迎來聽我唬爛、練肖話、講幹話。
