Mac OS Mavericks 使用筆記

我的黑金塔已經使用快兩個月了，現在除了做音樂還是需要開 Windows 7，還有網路銀行之外，都已經在 Mac OS 上做事了。這過程我把自己的 Mac OS 做了一些調整，還有找了很多工具，這些心得一一記錄在這裡，以備未來敗入 iMac or MBPR 時之需 XDDD

系統熱鍵

我是用 Windows 104 key 鍵盤，原本 Window Key 是對應到 Command
我參考了公司的 Macbook Pro 設定，把常用的快速鍵做了對應。

系統對應

  • Copy / Paste: Command + C / Command + V
  • Spotlight: Control + F4, 同事說這很好用，其實我很少用，大概跟我的思路習慣有關係。。。。。不夠懶ＸＤ
  • Screenshot:
  • Full screen: Command + Shift + 3
  • Window: Command + Shift + 4
  • 上述兩者，如果增壓 Alt, 那麼就是把圖檔放到 clipboard.
  • (常用) Remove file: Command + Backspace
  • (常用) 字典: 系統任一 Application 選取一單字, 同時按下 “Commnad + Control + D” 即可查詢單字
  • (常用) 顯示/隱藏 Dock: Alt + Command + D

桌面顯示

  • Launchpad: F4
  • Mission Control: F3
  • 顯示桌面: F11 (所有視窗都會跑到左右兩邊，顯示桌布)
  • 顯示 Dashboard: F12
  • 切換 desktop: Control + 左右方向鍵
  • 顯示目前應用程式最近開啟的文件: Alt + 下
  • 如果目前 Focus 在 Finder, 那就會顯示最近開啟的文件

文字編輯

在 Windows 上寫 Code 時，我習慣用 Home/End/PageDown/Up/Del 這些搭配 Control/Shift，不管是段落、還是快速在字或者行數中移動，有很高的工作效率，但是在 Mac 上卻無法如此，讓我在工作效率上有很大的折扣。特別是在 Macbook Pro 上是沒有 Home/End/PD/PU 等鍵，這些在 ThinkPad 上都有。

但是兩個月下來我也慢慢習慣 Mac 的用法了，整理和 Windows / Vi 比較如下：

Function Mac OS M$ Windows Vi / Vim
Jump word Alt + 方向鍵 Alt + 方向鍵 w / W
Home of line Command + 左 或者 Control + A Home 0
End of line Command + 右 或者 Control + E End $
Start of File Control + Home Control + Home gg
End of File Control + End Control + End G

以上如果要選取 “段落”、”字”、或者 “全選”，除了 vi，都可以搭配 Shift 使用。其他再慢慢補充。。。。

vi or vim 參考另一篇心得 [Vim-Notes][/2015/02/18/Vim-Notes/]

工具篇

過去很多工具在 Windows 已經有使用習慣的，整理一下目前使用的，大部份都是不用錢的。

系統工具類

  • VNC client on Safari: 在 URL 輸入 vnc://{vnc_server_ip} 就可以使用，這是 Mavericks 之後版本才有的
  • atMonitor: 在狀態列顯示系統詳細資訊，超好用！不用錢！
  • MPlayerX: 看電影、海賊王就靠它了！
  • VLC: 用來研究 streaming / RTSP
  • Remote Mouse: 用 iPhone / iPad 當滑鼠
  • TeamViewer: 修電腦，好人專用 QQ

開發工具

  • Sequel Pro: MySQL Client
  • SQLite Database Browser
  • svnX
  • GitX
  • zoc: ssh client

影像工具

自從 PhotoImpact 死了之後，PaintShop Pro 變得很複雜之後，我就很少在玩影像了。現在偶爾會做照片的批次處理而已，所以找了幾個免費的工具玩玩。另外比較需要的是看圖工具，Mac OS Mavericks 內建的 Preview 有些陽春，顯示速度也不快。

  • FireAlpaca: 簡易版的 PhotoShop，基本功能都有，免費，夠用了
  • JView: 看圖工具, Made in Taiwan

系統設定 / 調整

使用過程中，除了安裝的介紹之外，已經很少做系統參數調整了，寫 Blog 抓圖時才有調整。。。
