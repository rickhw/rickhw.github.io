Redmine Plugins and Themes

整理一些不錯的 Redmine Plugins and Themes。

Install and Unstall

Plugins 安裝

依照 Redmine 官方文件，大部分的 Redmine Plugin 安裝都很簡單，有兩種：

第一種：在 /[REDMINE_HOME]/plugins 執行 bundle

  1. 把 Plugin 複製到 /[REDMINE_HOME]/plugins
  2. 進入 plugins 目錄
  3. 執行: bundle exec rake redmine:plugins:migrate RAILS_ENV=production

第二種：在 /[REDMINE_HOME] 執行 bundle

  1. Easy Gantt: 不需要進入 plugin 目錄，而是直接在 /[REDMINE_HOME]/plugins 執行 bundle.
  2. User Mentions: 有些東西放 [REDMINE_HOME]，有些放 /[REDMINE_HOME]/plugins，
  3. 都在 [REDMINE_HOME] 執行: bundle exec rake redmine:plugins:migrate RAILS_ENV=production

刪除 Plugins

  1. Downgrade database schema, 記得更改 <plugin_name>, <version>
  2. 刪除 {REDMINE_HOME}/plugins 裡的 plugins
  3. 重啟 redmine

Redmine Plugins

推薦安裝

  • Redmine Dashboard: 很讚！
    • For Redmine 4.0.x，要直接從 Github Clone 才能安裝。
  • Easy Gantt plugin：免費不錯用，要錢的功能更完整
  • Redmine Agile plugin: 很棒，大推，要注意 Theme 的搭配。single version: $299 USD
  • Redmine Issue Templates plugin：有 Global and Project Level 可以定義。
    • Updated 2019/05: 新增了 Template for Notes，可以再回覆時，提供樣板
  • Redmine Slack：可以把 update 吐到 Slack Channel
  • Redmine User Mentions plugin
    • Updated 2019/05: For Redmine 4.0 的安裝步驟有些麻煩
  • Redmine Checklists plugin
  • Revision Branches: 更清楚顯示 git Branches
  • redmine_lightbox2: 顯示圖檔，不會獨立顯示
  • redmine_issue_view_columns: 在 Issue Related View，客制可以顯示欄位
  • Redmine Issues Tree: 動態顯示 Issue Tree，大推!

Plugins 評估過的，但不推薦

  • Redmine Bitbucket plugin
  • Redmine Workflow Enhancements
  • Advanced Road
  • Easy WBS plugin
  • Redmine People: 可安裝 Redmine 4.0.x

準備安裝

  • dmsf: Document Management System Features plugin: 網友推薦
  • Redmine draw.io: 網友推薦
  • Redmine PivotTable: 樞紐分析
  • Custom Menu
  • My Page
  • Magic My Page ($199)
  • Computed Custom Field
  • Deployer
  • Redmine Workflow Enhancements
  • Custom Workflows

已停用

  • Extra Query
  • Common lib
  • Global rule

比較複雜，未來再導入

有 Frontend 問題，會跟 Template 衝突，需要調整。

  • Redmine CKEditor plugin: http://github.com/a-ono/redmine_ckeditor
