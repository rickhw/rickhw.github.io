關於 作者

自許為 軟體開發者、音樂愛好者，工作角色經歷： Senior Software Developer、SQA Manager / SDET Lead、Operation / Infrastructure Manager、Architect @ 91APP、Product Manager / Tech Manager / Solution Architect @ TSMC

專長為軟體工程、系統架構、系統分析與設計、軟體測試、系統維運、產品設計、軟體開發、經營管理、AWS、DevOps / SRE。這幾年專注在大型 SaaS / Multi-tenancy / API Design、分散式系統、經營管理 等領域。

著有技術部落格：Complete Think、譯著：《分散式系統設計》 (2019)、共同著作：《軟體測試實務 I、II》 (2023/06)、《SRE 實踐與開發平台指南》 (2023/08)、共同著作：《軟體品質全面思維》 (2025)。

工作之餘喜歡金庸武俠、哲學、科幻小說、人文藝術。也是業餘音樂愛好者，涉略涵蓋 吉他、鍵盤、編曲、教學，著有音樂部落格：喝咖啡聊音樂 及 電子書。
