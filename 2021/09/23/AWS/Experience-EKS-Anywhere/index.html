Experience EKS Anywhere

AWS жҠҠ EKS жң¬иә«й–ӢжәҗдәҶпјҢзЁұзӮә EKS AnywhereпјҢйҷӨдәҶеңЁ AWS иҮӘе·ұд№ӢеӨ–пјҢд№ҹеҸҜд»Ҙе®үиЈқеңЁз§Ғжңү (on-premises) зҡ„з’°еўғгҖӮ

жң¬ж–Үж•ҙзҗҶй«”й©—зҡ„зӯҶиЁҳпјҢжҲ‘еңЁ macOSгҖҒubuntu20.04 йғҪжңүеҒҡйҒҺпјҢжң¬ж–ҮиЁҳйҢ„еүҮд»Ҙ ubuntu 20.04 зӮәдё»пјҢе…§е®№д»ҘеҸғиҖғ EKS Anywhere е®ҳж–№ж–Үд»¶зӮәдё»гҖӮ

жә–еӮҷ

еүҚзҪ®дҪңжҘӯ

жҲ‘з”Ё VMWare жә–еӮҷдәҶдёҖеҸ° Ubuntu 20.04пјҢиіҮжәҗжңү:

  • CPU: 4core
  • Memory: 16GiB
  • Disk: 100GiB
  • Network: Bridge Mode

и»ҹй«”жә–еӮҷпјҡ

  • Docker CE v20.x д»ҘдёҠпјҢзӣёй—ңе®үиЈқеҸҜд»ҘеҸғиҖғ K8s еӯёзҝ’зӯҶиЁҳ - kubeadm жүӢеӢ•е®үиЈқ

еҰӮжһңжҳҜзӣҙжҺҘеңЁ macOS дёҠз”Ё Docker CE и·‘пјҢе»әиӯ°е…Ҳдҝ®ж”№ docker Resources иіҮжәҗпјҢжҲ‘жҠҠиЁҳжҶ¶й«”жӢүеҲ°еҫҲй«ҳпјҢеҗҢжҷӮй—ңжҺүе…¶д»–зҡ„ containersпјҢеҰӮең–пјҡ

жә–еӮҷ K8s & EKS CLI

EKS Anywhere дё»иҰҒдҪҝз”Ё е…©еҖӢ AWS зҡ„ CLI:

  1. eksctl
  2. eksctl-anywhere

дёӢијүйҖҷе…©еҖӢе·Ҙе…·пјҢйҖҷеҖӢзӯҶиЁҳзөұдёҖйғҪж”ҫеңЁ $HOME/bin еә•дёӢгҖӮ

ж–°зҡ„ VM д№ҹиҰҒжә–еӮҷ kubectlгҖӮ

дёӢијү docker images

EKS anywhere е®үиЈқйҒҺзЁӢпјҢйңҖиҰҒдҪҝз”ЁдёҖдәӣ EKS е®ҳж–№зҡ„ ImagesпјҢеҰӮжһңз¶Іи·ҜйҖҹеәҰдёҚеҝ«пјҢе®үиЈқйҒҺзЁӢйңҖиҰҒжҷӮй–“зӯүеҫ…пјҢжүҖд»Ҙе»әиӯ°е…ҲжҠҠйҖҷдәӣ images дёӢијүгҖӮ

жҲ‘е®үиЈқзҡ„жҷӮеҖҷпјҢдёӢијүдәҶд»ҘдёӢ images:

еә•дёӢжҳҜз”Ё docker дёӢијүзҡ„ scriptsпјҢйҖҷдәӣиіҮиЁҠжҳҜеңЁ ~/${CLUSTER_NAME}/generated/clusterctl_tmp.yaml иЈЎйқўжүҫеҲ°зҡ„пјҢдёҰдё”ж•ҙзҗҶеҮәдҫҶзҡ„пјҢйҖҷдәӣиіҮиЁҠжҮүи©ІжңғйҡЁи‘—зүҲжң¬зҡ„жј”йҖІиҖҢжңүжүҖж”№и®ҠпјҢжүҖд»Ҙд»ҘдёӢзҡ„иіҮиЁҠиҲҮзүҲжң¬еғ…дҫӣеҸғиҖғпјҡ

дёҠиҝ°иіҮиЁҠжҳҜжҲ‘иҮӘе·ұжүӢеӢ•ж•ҙзҗҶзҡ„пјҢжҮүи©Іжңғи®ҠеҢ–еҫҲеҝ«гҖӮд№ӢжүҖд»Ҙж•ҙзҗҶйҖҷд»ҪпјҢжҳҜжҲ‘еңЁ lab зҡ„йҒҺзЁӢпјҢеүӣеҘҪйҒҮеҲ°з¶Іи·ҜзӢҖжіҒдёҚеҘҪпјҢжүҖд»ҘиҠұеҫҲеӨҡжҷӮй–“дёӢијү images пјҢеҰӮжһңй җе…ҲдёӢијүеҘҪпјҢж•ҙеҖӢе®үиЈқйҖҹеәҰе°ұеҝ«еҫҲеӨҡгҖӮ

дёҠиҝ°иіҮиЁҠжҳҜе®Ңе…ЁжІ’жңүиӘҝж•ҙ docker CE зҡ„иіҮжәҗ (й җиЁӯиЁҳжҶ¶й«”жҳҜ 8GiB) гҖҒи·‘еңЁ MacOS дёҠпјҢйҒҺзЁӢи·‘дәҶе°Үиҝ‘ 10 еҲҶйҗҳпјҢиӘҝж•ҙеҲ° 20GiB д№ӢеҫҢпјҢи·‘е®Ңзҙ„ 7 еҲҶйҗҳгҖӮ

еҰӮжһңз•«йқўеҫҲд№…жІ’еҸҚжҮүпјҢеҸҜд»ҘеўһеҠ еҸғж•ё -v 5 дәҶи§ЈзӢҖжіҒпјҢеә•дёӢжҳҜеңЁ Ubuntu дёҠеҹ·иЎҢжҷӮзҡ„зҙҖйҢ„пјҡ

йҖҷж®ө log еҸҜд»ҘзңӢеҲ°йҒҺзЁӢи©ізҙ°зҡ„еҹ·иЎҢиіҮиЁҠпјҢж•ҙеҖӢйҒҺзЁӢи·‘дәҶеӨ§зҙ„ 8 еҲҶйҗҳпјҢйҖҷжҳҜжҲ‘е·Із¶“й җе…ҲдёӢијү docker images зҡ„зӢҖжіҒпјҢжүҖд»Ҙе»әиӯ°е…ҲдёӢијүйҒҺ docekr images.
